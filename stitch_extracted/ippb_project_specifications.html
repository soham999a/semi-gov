# IPPB WEBSITE - IMPLEMENTATION PROMPT ## 🎯 OBJECTIVE Build a complete, production-ready IPPB (India Post Payments Bank) Service Center Website using Firebase as backend and HTML/CSS/JavaScript frontend following the design specification document exactly. ## ⚠️ CRITICAL RULES - MUST use Firebase (Authentication, Firestore, Storage, Cloud Functions, Hosting) - MUST follow color palette exactly - use CSS variables, never hardcode colors - MUST implement all responsive breakpoints - mobile-first approach - MUST achieve WCAG AA accessibility - MUST match typography scales - MUST implement all animations - MUST test on all browsers - MUST validate all forms - MUST handle errors gracefully - MUST optimize performance - Lighthouse score 90+ ## 🎨 DESIGN SPECIFICATIONS ### Color Palette - Primary Blue: #1B3A7D - Success Green: #2ECC71 - Alert Orange: #E67E22 - Danger Red: #E74C3C - Teal Accent: #16A085 - Gold Accent: #F39C12 - Dark Gray: #2C3E50 - Medium Gray: #7F8C8D - Light Gray: #ECF0F1 - Very Light Gray: #F8F9FA - White: #FFFFFF ### Typography - H1: 48px, weight 700 - H2: 36px, weight 700 - H3: 28px, weight 600 - H4: 24px, weight 600 - H5: 20px, weight 600 - Body: 16px, weight 400 - Small: 14px, weight 400 - Button: 16px, weight 600 - Label: 14px, weight 600 ### Spacing - xs: 4px, sm: 8px, md: 16px, lg: 24px, xl: 32px, 2xl: 48px, 3xl: 64px ## 📱 PAGE SPECIFICATIONS 1. **HOME PAGE**: Navbar, Hero, Stats, Services Grid, Why Choose Us, How It Works, Testimonials, CTA, Footer. 2. **SERVICES PAGE**: Sidebar nav, 6 categories (Banking, AEPS, Gov Schemes, Insurance, Investment, Business). 3. **LOGIN PAGE**: Phone + OTP flow. 4. **OTP PAGE**: 6-digit input, timer. 5. **REGISTRATION**: 3-step form (Basic, Address, Account Type). 6. **CUSTOMER DASHBOARD**: Accounts, Transactions, Requests. 7. **ADMIN DASHBOARD**: User & Request management. 8. **CONTACT PAGE**: Form, Map, Info.
